Latest Projects

Verbugata

Practice Portuguese verb conjugations

An application to help learn Portuguese verb conjugations. I made this as learning verb conjugations can be a difficult part of language learning, and I couldn't find a tool that just focused on verbs. It is written in React and uses Google's Material UI library.

Verbugata has been well received by the Portuguese language learning community. I am in the process of building a back-end with user management to store progress. I would love to expand this to other Latin languages as well!
